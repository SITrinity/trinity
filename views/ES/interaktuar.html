Objetivos

  • Desarrollar un sistema de entrenamiento y capacitaciones que permita a los usuarios interactuar con ambientes virtuales, mediante el uso de tecnología emergente, con el objetivo de facilitar el proceso de enseñanza-aprendizaje dentro de las organizaciones.
  • Expandir el uso de las tecnologias emergentes para los procesos de aprendizaje aplicados en la tranferencia de conocimientos en las ARL.
  • Desarrollar una serie de aplicaciones moviles y web para el uso de Realidad Aumentada, Realidad Virtual y Simulación Inmersiva.
